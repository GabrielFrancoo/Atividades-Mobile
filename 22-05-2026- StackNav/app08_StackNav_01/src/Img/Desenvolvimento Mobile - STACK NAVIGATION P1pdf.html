STACK NAVIGATON - PARTE 1

INTRODUÇÃO

Nesta  primeira parte vamos tratar do Stack Navigation  ou,  Navegação de Pilha no react native. Iniciaremos com um exemplo simples e depois vamos aumentando sua complexidade, alias, como sempre fizemos.

Conforme destacado pelo site React Navigation, a utilização do navegador por pilha fornece uma maneira para seu aplicativo fazer a transição entre telas, onde cada nova tela é colocada no topo de uma pilha. Apontam também que, por padrão, o stack navigator é configurado para ter a aparência e comportamento familiares do iOS e do Android: novas telas deslizam da direita no iOS, usam animação padrão do SO no Android. 

Observação: As animações podem ser personalizadas para atender necessidades específicas de cada app.

Vamos lá...

Agora que o projeto está criado, vamos em frente...

3 - INSTALANDO AS BIBLIOTECAS

  • Para instalar as bibliotecas necessárias, digitamos o seguinte comando:

npx expo install @react-navigation/native 

Quando você cria um app simples com apenas um componente App, não precisa dela.  Mas, quando o aplicativo começa a ter várias telas — por exemplo:  Login // Home // Cadastro // etc

… você precisa de uma forma organizada de trocar de tela, voltar para a tela anterior,  passar dados entre telas,  controlar pilha de navegação, etc;

É exatamente isso que esta biblioteca que estamos instalando vai permitir fazer.



4 - INSTALANDO DEPENDÊNCIAS

Logo que a instalação acabar, execute o comando abaixo, no mesmo diretório do seu projeto (no prompt do DOS ou no Terminal so VSCode):

npx expo install react-native-screens react-native-safe-area-context 

A primiera dependência permite otimizar as telas nativas, melhorando a performance do app e a segunda permite apresentar conteúdo apenas em áreas seguras da tela do dispositivo. 

5 - NAVEGANDO POR PILHAS (STACK NAVIGATION)

Como você já sabe, neste subTópico vamos aprender a trabalhar com navegação enter telas com StackNavigation, onde as telas são empilhadas e desempilhadas, como uma pilha de memória FILO (First In, Last Out).

Para conseguir utilizar esta maneira de navegar entre telas, é necessário instalar a biblioteca apropriada (dentro do seu projeto), como mostra a linha abaixo:

npx expo install @react-navigation/native-stack

COMO FUNCIONA A NAVEGAÇÃO ENTRE TELAS

A primeira coisa que você precisa ter em mente é que o arquivo App.js não poderá ser uma tela normal, mas sim uma tela na qual a pilha de telas e a rota de navegação é definida.

Sendo assim, precisamos importar algumas bibliotecas para poder gerenciar a navegação, as bibliotecas são:

  • NavigationContainer :

Permite que usemos um Container (uma caixa) dentro do qual serão definidas as telas navegáveis;

  • createNativeStackNavigator : 

Permite utilizar a navegação por pilhas. Permite utilizar uma função de mesmo nome que retorna um objeto contendo 2 propriedades: Screen e Navigator. Ambos são componentes React usados para configurar o navegador. O Navigator deve conter elementos de Screen como seus filhos para definir a configuração das rotas.
