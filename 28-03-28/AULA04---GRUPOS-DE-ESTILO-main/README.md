# App StyleSheet

Projeto desenvolvido na disciplina de Desenvolvimento Mobile.

## Descrição
Aplicativo mobile desenvolvido com React Native e Expo, 
replicando a interface de um app de saúde com cabeçalho, 
corpo com scroll e rodapé de navegação.

## Tecnologias utilizadas
- React Native
- Expo
- react-native-web
- StyleSheet

## Como rodar
1. Clone o repositório
2. Execute `npm install`
3. Execute `npm run web`
