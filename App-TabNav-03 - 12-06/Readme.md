# 🏋️ Academia App — Tab Navigation

App mobile desenvolvido em React Native com Expo, utilizando navegação por abas (Tab Navigation) como parte do Desafio 03 da disciplina de Desenvolvimento Mobile.

---

## 📱 Telas

| Aba | Descrição |
|-----|-----------|
| 💪 Exercícios | Lista do treino do dia com grupo muscular e séries |
| 🥗 Dieta | Plano alimentar com horários e refeições |
| 👤 Perfil | Dados do aluno, estatísticas e conquistas |

---

## 🛠️ Tecnologias

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [@react-navigation/bottom-tabs](https://reactnavigation.org/)
- [@expo/vector-icons](https://icons.expo.fyi/)

---

## 🚀 Como executar

```bash
# Instalar dependências
npx expo install @react-navigation/native
npx expo install react-native-screens react-native-safe-area-context
npx expo install @react-navigation/bottom-tabs
npx expo install @expo/vector-icons

# Rodar no navegador
npm run web
```
